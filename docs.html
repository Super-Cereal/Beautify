Documentation

BaseUrl: http://127.0.0.1:5000/api/v.1.0

Authentification

/auth

get

Узнает, залогинен ли текущий пользователь

{
  "resultCode": (boolean),
  "data": {
    "id": (number), 
    "email": (string)
  }
}

resultCode: 0 или 1, если пользователь залогинен или нет
id: id текущего пользователя или Null
email: email текущего пользователя или Null

post

Позволяет войти в аккаунт

{
  "email": required(string),
  "password": required(string),
  "remember_me": (boolean)
}               
{ 
  "resultCode": (boolean),
  "data": {
    "userId": (number)
  }
}

resultCode: 0 или 1 успешна операция или нет
userId: id пользователя, либо Null

delete

Разлогиневает пользователя

{
  "resultCode": (boolean),
}

resultCode: 0 или 1, если операция прошла успешна или нет
