BaseUrl: http://127.0.0.1:5000/api/v.1.0

Authentification

/auth

get

Узнает, залогинен ли текущий пользователь

Headers:
"Authorization": "Bearer ‹access_token›"

access_token: токен, выдаваемый при регистрации или логине 

{
  "resultCode": required(boolean),
  "message": (string)
  "data": {
    "id": required(number),
    "nickname": required(number),
    "email": required(number),
  }
}

resultCode: 0 или 1, если пользователь залогинен или нет
data Содержит инфу о текущем пользователе
message: Содержит ошибку если resultCode = 1

post

Позволяет войти в аккаунт

{
  "email": required(string),
  "password": required(string)
}               
{ 
  "resultCode": required(boolean),
  "message": (string),
  "data": {
    "id": required(number),
    "nickname": required(number),
    "email": required(number),
    "access_token": required(string)
  }
}

resultCode: 0 или 1 успешна операция или нет
message: Содержит ошибку если resultCode = 1
data Содержит инфу о текущем пользователе
data.access_token: Токен авторизации для пользователя

Users list resourses

/users

get

Получает инфу о всех пользователях

{
  "resultCode": required(boolean),
  "message": (string)
  "data": {
    "users": [
      {
        "id": required(number),
        "nickname": required(number),
        "email": required(number)
      }
    ]
  }
}

resultCode: 0 или 1, если пользователь залогинен или нет
message: Содержит ошибку если resultCode = 1
data.users: Содержит массив пользователей с указанной информацией

post

Позволяет зарегистрироваться

{
  "nickname": required(string),
  "email": required(string),
  "password": required(string)
}               
{ 
  "resultCode": required(boolean),
  "message": (string),
  "data": {
    "id": required(string),
    "nickname": required(string),
    "email": required(string)
    "access_token": required(string)
  }
}

resultCode: 0 или 1 успешна операция или нет
message: Содержит ошибку если resultCode = 1
data: Содержит инфу о текущем пользователе
data.access_token: Токен авторизации для пользователя
